## Drescription

A responsive portfolio built with Next.js and Tailwind CSS to present my work, including web applications, machine learning projects, and AI solutions.

## Link to Vercel

[jeferson-delrio.vercel.app](https://jeferson-delrio.vercel.app)

## Getting Started

First, Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

This project follows the Atomic Design methodology:

-- Atoms → Basic UI elements (buttons, text, icons)

-- Molecules → Groups of atoms forming components

-- Organisms → Full sections of the page

-- Pages → Final layout and composition

## Tech Stack

-- Next.js
-- React
-- Tailwind CSS
